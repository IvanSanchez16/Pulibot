# Pulibot
Pulibot es un bot para discord elaborado con Python apoyandose de la libreria discord.py
#Instalación
Necesitas de pipenv para instalar las dependencias de una manera más sencilla.

```bash
pip install pipenv
```
Después instalaremos todas las dependencias con

```bash
pipenv install
```
Por utlimo ocuparemos crear un archivo .env para los access-tokens necesarios para que el bot funcione
